# Candles (OHLCV)

```
GET /v1/candles/{market_uid}
```

Outcome-level OHLCV per bar with VWAP, volume, and open interest.

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `market_uid` | string | Canonical market identifier |

## Query Parameters

All parameters are optional.

| Parameter | Type | Description |
|-----------|------|-------------|
| `interval` | enum | Bar interval: `1m`, `5m`, `15m`, `1h`, `4h`, `1d` |
| `start` | string | Start time filter (ISO-8601 UTC timestamp) |
| `end` | string | End time filter (ISO-8601 UTC timestamp) |
| `limit` | integer | Results per page (default: 500, max: 5000) |
| `cursor` | string | Opaque cursor from previous `next_cursor` |

## Example Request

Get 5-minute bars for MLB rivalry game:

```bash
curl -s -H "Authorization: Bearer $TOKEN" \
  "https://api.predexon.com/v1/candles/mkt_evt_mlb_2025_06_24_NYY_BOS_yesno?interval=5m&start=2025-06-24T22:00:00Z&end=2025-06-25T02:00:00Z&limit=500"
```

## Response Fields

| Field | Description |
|-------|-------------|
| `bar_spec` | Specification for bar alignment and boundaries |
| `bar_spec.timezone` | Timezone for bar boundaries (always `UTC` in MVP) |
| `bar_spec.alignment` | Bar alignment semantics (`left_closed_right_open`) |
| `bar_spec.boundary` | Boundary type (`calendar` for standard intervals) |
| `bar_spec.fields` | Fields included in each bar |
| `candles` | Array of OHLCV bars |
| `candles[].ts` | Bar timestamp (start of interval) |
| `candles[outcome]` | OHLCV data per outcome |
| `vwap` | Volume-weighted average price |
| `oi` | Open interest at bar close |

## Notes

- **Bar semantics**: Bars follow left-closed, right-open semantics in UTC (e.g., `23:10:00` includes trades from `23:10:00.000` up to but not including `23:15:00.000`).
- **Pagination**: Use `cursor` for long time ranges to avoid hitting response size limits.
- **Per-outcome data**: Each outcome (YES, NO, etc.) has its own OHLCV data.
