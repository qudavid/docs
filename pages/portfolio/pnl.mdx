# PnL

```
GET /v1/portfolio/pnl
```

Time-series PnL showing realized, unrealized, and fee drag.

## Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `granularity` | enum | Time granularity: `day`, `week`, `month`, `year`, `all` |
| `start` | string | Start date (ISO-8601 UTC timestamp) - optional |
| `end` | string | End date (ISO-8601 UTC timestamp) - optional |

## Response Schema

```json
{
  "granularity": "day",
  "start": "2025-05-01",
  "end": "2025-10-06",
  "curve": [
    {
      "ts": "2025-05-01T00:00:00Z",
      "realized": "22.40",
      "unrealized": "8.10",
      "fees": "-0.40",
      "pnl_to_date": "30.10"
    }
  ],
  "breakdown": {
    "by_venue": {
      "POLYMARKET": "210.7",
      "KALSHI": "95.3"
    }
  }
}
```

## Example Request

Get daily PnL during Fed rate decision trading window:

```bash
curl -s -H "Authorization: Bearer $TOKEN" \
  "https://api.predexa.com/v1/portfolio/pnl?granularity=day&start=2025-06-01&end=2025-09-30"
```

## Response Fields

| Field | Description |
|-------|-------------|
| `granularity` | Time granularity of the PnL curve |
| `start` | Start of the reporting period |
| `end` | End of the reporting period |
| `curve` | Array of PnL data points |
| `curve[].ts` | Timestamp for this data point |
| `curve[].realized` | Realized PnL (from closed positions) |
| `curve[].unrealized` | Unrealized PnL (from open positions) |
| `curve[].fees` | Total fees paid (negative value) |
| `curve[].pnl_to_date` | Cumulative PnL up to this point |
| `breakdown` | PnL breakdown by various dimensions |
| `breakdown.by_venue` | Total PnL per venue |

## Notes

- **Realized PnL**: Profit/loss from positions that have been closed.
- **Unrealized PnL**: Mark-to-market P&L on open positions.
- **Fees**: Trading fees, gas costs, and other charges (shown as negative).
- **Cumulative**: `pnl_to_date` is the running total of `realized + unrealized + fees`.
