# Portfolio & Linking

Link native venue accounts and query normalized balances, positions, PnL, and activities; subscribe to live portfolio deltas.

## Base URLs

- **REST**: `https://api.predexa.com/v1`
- **WebSocket**: `wss://stream.predexa.com/v1`

## Authentication

All portfolio endpoints require authentication. Include your API token in the `Authorization` header.

## Conventions

For complete conventions (time format, numeric precision, pagination, IDs, error handling), see the [Discovery API Overview](/pages/discovery/overview).

## What's Included

### REST Endpoints
- **Balances** - Cash and collateral by venue
- **Positions** - Normalized outcome-level holdings with live marks
- **PnL** - Realized, unrealized, and fees over time
- **Activities** - Unified ledger of trades, deposits, withdrawals, resolutions

### WebSocket
- **Stream Portfolio** - Live positions and PnL deltas

### Account Management
- **Link Accounts** - Verify and link native venue accounts
