# Portfolio & Linking
<Callout type="warning" title="Experimental">
This endpoint group is experimental and may change without notice. Access is limited; contact support if you need early integration.
</Callout>

Manage venue account linkage and query normalized balances, positions, PnL, and activities. Subscribe to real-time portfolio deltas after linking.

## Base URLs

- **REST:** `https://api.predexon.com/v1`
- **WebSocket:** `wss://stream.predexon.com/v1`

## Conventions

Portfolio endpoints follow the same authentication, ID, and pagination conventions described in the [Discovery API Overview](/pages/discovery/overview). Additional notes:

- Link accounts via `/v1/link/wallets` before calling portfolio endpoints.
- All monetary values are returned as strings for precision.
- WebSocket streams use snapshot-then-delta semantics with monotonic `sequence` integers.

## Workflow

1. **Link accounts:** Verify venue accounts using [Link Accounts](/pages/portfolio/link-accounts).
2. **Fetch snapshots:** Pull [Balances](/pages/portfolio/balances), [Positions](/pages/portfolio/positions), and [PnL](/pages/portfolio/pnl) as needed.
3. **Review history:** Inspect [Activities](/pages/portfolio/activities) to reconcile trades and cash movements.
4. **Stream updates:** Subscribe to [Stream Portfolio](/pages/portfolio/stream) for live position and PnL deltas.
