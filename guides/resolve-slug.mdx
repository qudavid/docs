# Map Polymarket slug to market_uid

## Overview

Convert Polymarket slugs (or Kalshi tickers) to canonical `market_uid` values.

## Using /v1/resolve

Direct mapping from venue-specific identifiers to canonical IDs.

```bash
curl -X POST https://api.dome.plus/v1/resolve \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "venue": "polymarket",
    "slug": "will-trump-win-2024"
  }'
```
