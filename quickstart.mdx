# Quickstart

Get your first API calls working in 10 minutes.

## Prerequisites

- API key (contact us for access)
- Base URL: `https://api.predexon.com`
- curl, Node.js, or Python environment

## Step 1: Resolve native IDs

Map venue-specific identifiers to canonical `market_uid` values.

```bash
curl -X POST https://api.predexon.com/v1/resolve \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"venue": "polymarket", "slug": "will-biden-win-2024"}'
```

Response structure and examples will be documented here.

## Step 2: List markets

Fetch all markets for an event.

```bash
curl https://api.predexon.com/v1/markets?event_uid=EVENT_UID \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Step 3: Fetch quotes

Get the current best bid/ask/last for a market.

```bash
curl https://api.predexon.com/v1/quotes/MARKET_UID \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Step 4 (Optional): Order book snapshot

Retrieve L2 order book depth.

```bash
curl https://api.predexon.com/v1/orderbook/MARKET_UID \
  -H "Authorization: Bearer YOUR_API_KEY"
```
