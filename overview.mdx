# Overview

## Unify Polymarket + Kalshi with one API

Access events, markets, quotes, order books, trades, candles, tick data, and portfolio data across multiple prediction market venues with a single, normalized API.

## Why Predexa

- **Cross-venue normalization** — consistent data models across Polymarket and Kalshi
- **Fewer integrations** — one API instead of managing multiple venue-specific endpoints
- **Consistent models** — unified event, market, and outcome schemas

## What you can build in a day

- Price widgets
- Live tickers
- Sports dashboards
- Portfolio views
