# Welcome to Predexon

Predexon is a unified data API for prediction markets. Access normalized market data from Polymarket and Kalshi through a single integration.

## What Predexon provides

**Discovery**
- Search events and markets across venues
- Resolve venue-native IDs (Polymarket slugs, Kalshi tickers) to canonical identifiers
- Match equivalent markets across platforms

**Live Market Data**
- Real-time quotes (bid/ask/last)
- Trade history with pagination
- L2 order book snapshots
- WebSocket streams for quotes and order book updates

**Historical Data**
- OHLCV candles at multiple intervals
- Tick-level price history
- 24h volume and VWAP

**Portfolio Tracking** (optional)
- Link Polymarket and Kalshi accounts
- View positions and balances across venues
- Track P&L and activity history
- Real-time portfolio updates via WebSocket

## Why use Predexon

- **One API, multiple venues** — integrate once, access Polymarket and Kalshi
- **Normalized data models** — consistent schemas across all venues
- **Production-ready** — designed for reliability and performance
- **Developer-friendly** — clean REST endpoints, WebSocket streams, and comprehensive documentation
