# Predexon API Documentation

Welcome to Predexon - a unified data API for prediction markets. Access normalized market data from **Polymarket** and **Kalshi** through a single integration.

## Why Predexon?

Building on multiple prediction market platforms means dealing with different APIs, data formats, and authentication methods. Predexon solves this by providing:

- **Unified Schema** - One consistent data model across all venues
- **Multi-Venue Access** - Integrate once, access Polymarket and Kalshi
- **Real-Time & Historical** - Live WebSocket feeds and historical OHLCV data
- **Production-Ready** - Built for reliability, performance, and scale
